HCA Tuatara: presubmission data tracking

About

Tuatara is a database to keep track of projects that the HCA is integrating or interested in integrating into the Atlas. It can answer data roadmap questions like "What is the current status of single nuclei-seq? I'd like to know when we can expect it so I can prioritize pipeline development, and from who so I can read papers from the labs." Tuatara is built on top of Python, Django, and a relational database that we hope will replace several spreadsheets.

Access

If you are interested in access to Tuatara through a prototypical web interface, please email clayfischer@ucsc.edu. We will be rolling out broader access as we interate through feedback.

API

A RESTful API allows one to query for current status of projects. Combined with the tracker.data.humancellatlas.org API, a full dashboard can be built to show all data known and in progress. Not all data is available through the API due to GDPR constraints, but enough data to make informed decisions based on known projects is available. Here's a list of the restful urls supported by the API:

  • https://tuatara.data.humancellatlas.org/api - lists all classes and the number of objects in each class
  • https://tuatara.data.humancellatlas.org/api/project - lists information on all projects
  • https://tuatara.data.humancellatlas.org/api/project/[project short name] - list info on project with short name
  • https://tuatara.data.humancellatlas.org/api/projectstatus - List and describe values in project.status field
  • https://tuatara.data.humancellatlas.org/api/contributor - List contributors to any project
  • https://tuatara.data.humancellatlas.org/api/tracker - Post-submission tracking of project
  • https://tuatara.data.humancellatlas.org/api/cdnalibraryprep - Types of single cell RNA sequencing assay
  • https://tuatara.data.humancellatlas.org/api/disease - Diseases currently in any project.
  • https://tuatara.data.humancellatlas.org/api/organ - Organs currently used in any project.
